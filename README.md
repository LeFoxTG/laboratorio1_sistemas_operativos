# laboratorio1_sistemas_operativos

## Funcionamiento

1. Abrir la carpeta en Visual Studio Code

![abrir carpeta vscode](https://i.ibb.co/zwjTN3K/imagen-2026-03-11-124836187.png)

2. Instalar la extension "Live Server"

![extension live server](https://i.ibb.co/MyFnjB3T/imagen-2026-03-11-125941577.png)

3. En app.js, en la parte inferior derecha presionar el botón "Go Live"

![go live](https://i.ibb.co/LD8d2yhQ/imagen-2026-03-11-130248675.png)

4. Se abrirá el navegador predeterminado y el simulador estará listo para su uso

![simulador funcionando](https://i.ibb.co/TBVfys8k/imagen-2026-03-11-130350939.png)
